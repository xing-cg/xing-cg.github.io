常用命令

命令标识 含义
l 显示代码
Enter（回车键） 重复上一条命令
b 行号 为某行添加断点
info break 查看断点信息(bnum)
delete bnum 删除断点对应的编号
r、run 启动程序
n、next 单步执行
p 变量名、print 变量名 打印变量名内容
q 退出调试
s、step 进入函数
f、finish 跳出函数
continue 继续程序（到下一个断点）
