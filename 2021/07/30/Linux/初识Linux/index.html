初识Linux

Created2021-07-30Updated2022-04-18

Linux学习内容

课程周期：一周两次课，2~3月结束

  1. 基本操作
  2. gcc/g++（程序如何编译）
  3. 工程文件管理makefile
  4. 调试gdb
  5. 库文件
  6. 进程：复制进程fork、替换进程
  7. 进程间通信（QQ互发信息）：管道、共享内存、信号列
  8. 线程：同步、并发
  9. 网络编程：跨主机发送数据（最好有计算机网络基础），TCP、HTTP、I/O复用方法

初识内容

  1. 目录结构：Linux中的文件夹称为目录
  2. 文件类型
  3. 权限
  4. 命令

总结

  1. 与Windows相比：Linux多用命令行，我们要掌握基本的命令。
  2. 了解系统的目录结构，是个倒状树，记住常见的四五个目录的作用。
  3. 用户存放文件，要放到自己的家目录。

Linux目录结构

目录 描述
/ 根目录
bin 二进制可执行程序
dev 设备文件，把硬件设备抽象成为了文件
lib 库文件
mnt 临时挂载移动设备
proc 将进程的信息映射到了文件中
home 普通用户的家目录，以用户名为准，如/home/xcg
root 管理员的家目录

boot

存放系统内核，系统启动相关

bin

存放命令

etc

系统的配置文件

lib

库文件

home

所有普通用户的家目录

文件类型
普通文件（文档） -
目录文件（文件夹） d
管道文件 p
设备文件 c,b
链接文件 l
套接字文件 s
